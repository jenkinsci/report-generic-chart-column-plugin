Specify additional files to copy during the chart generation process.

This field accepts a comma-separated list of file paths or patterns that should be copied when the Generic Chart Publisher performs its operations. The files will be copied from the build workspace to the locations specified in the "Target folders" field.

Examples:

  • report.html - Copy a single file
  • *.log, *.txt - Copy all log and text files
  • results/*.json - Copy all JSON files from the results directory

Leave empty if no additional files need to be copied.
