Specify the additional files which should be copied. By default, only files generated by this plugin are exported.

This field accepts a comma-separated list of file. Note that build.xml may not yet exists.

Examples:

  • ../../config.xml - config
  • archive/reports - copy some artifacts
  • /backup,/archive -copy two absolute files (dirs are nto supported)

Leave empty if no additional files need to be copied.
